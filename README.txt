
	IMPORTANTE:

	- Descomprimir la carpeta dentro del directorio de su servidor web local (XAMPP o WampServer)

	- En el navegador web escribir: localhost/phpmyadmin para crear la base de datos en MySQL con la siguiente estructura

	Estructura de la base de datos
	- Nombre de la base: listaContactos
	- nombre de la tabla: datos
	- Nombre de los campos:
						- id INT (4) AUTO_INCREMENT
						- nombre VARCHAR (50)
						- contacto VARCHAR (50)

	PD: Si surge alguna duda, editar y leer el archivo db.php

	- En otra pestaña del navegador web escribir: localhost/[nombre_de_la_carpeta], esto iniciará el script interpretando el archivo index.php

	Por ejemplo, localhost/login
